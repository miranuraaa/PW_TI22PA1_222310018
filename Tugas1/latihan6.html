    • Python
    Python adalah bahasa pemrograman tingkat tinggi yang serbaguna, yang pertama kali dirilis pada tahun 1991 oleh Guido van Rossum. Ia dirancang dengan fokus pada keterbacaan kode dan produktivitas dalam pengembangan perangkat lunak.

    JavaScript
    JavaScript adalah bahasa pemrograman yang digunakan terutama untuk mengembangkan aplikasi web interaktif. Ini adalah bahasa pemrograman berbasis klien yang dieksekusi di browser pengguna dan sering digunakan untuk menambahkan interaktivitas pada halaman web.

    Java
    Java adalah bahasa pemrograman tingkat tinggi yang populer untuk pengembangan perangkat lunak. Ini sering digunakan untuk mengembangkan aplikasi perangkat lunak berbasis desktop, aplikasi web, perangkat lunak server, permainan, dan aplikasi seluler.

    C#
    C# adalah bahasa pemrograman yang dikembangkan oleh Microsoft. Ini sering digunakan untuk mengembangkan aplikasi berbasis Windows, aplikasi permainan menggunakan platform Unity, aplikasi web menggunakan ASP.NET, dan aplikasi seluler menggunakan platform Xamarin.

    C
    C adalah bahasa pemrograman yang sangat populer dan berpengaruh. Ini adalah bahasa pemrograman prosedural yang cepat dan efisien yang digunakan untuk mengembangkan berbagai jenis perangkat lunak, termasuk sistem operasi, perangkat lunak aplikasi, dan perangkat lunak embedded.

    C++
    C++ adalah bahasa pemrograman yang kuat dan efisien yang digunakan untuk pengembangan perangkat lunak sistem, perangkat lunak aplikasi, permainan, perangkat lunak embedded, dan banyak lagi. Ini adalah perluasan dari bahasa C dengan penambahan fitur-fitur seperti pemrograman berorientasi objek.

    Go
    Go, atau disebut juga Golang, adalah bahasa pemrograman yang dikembangkan oleh Google. Ini dirancang untuk menjadi bahasa yang sederhana, efisien, dan mudah digunakan untuk pengembangan perangkat lunak sistem, perangkat lunak server, dan perangkat lunak web.

    R
    R adalah bahasa pemrograman dan lingkungan perangkat lunak yang digunakan terutama untuk analisis statistik dan visualisasi data. Ini sangat populer di antara para ilmuwan data dan analis data untuk melakukan analisis data yang kompleks.

    Swift
    Swift adalah bahasa pemrograman yang dikembangkan oleh Apple untuk pengembangan aplikasi iOS, macOS, watchOS, dan tvOS. Ini dirancang untuk menjadi lebih aman, lebih mudah dibaca, dan lebih cepat daripada bahasa pemrograman sebelumnya yang digunakan untuk pengembangan aplikasi Apple, seperti Objective-C.

    PHP
    PHP adalah bahasa pemrograman yang sering digunakan untuk mengembangkan aplikasi web dinamis. Ini adalah bahasa pemrograman sisi server yang digunakan untuk menghasilkan konten dinamis pada halaman web, seperti halaman login, formulir, dan halaman berita.
